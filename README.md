# Dotfiles

## How to install

```bash
$ bash -c "$(curl -fsSL raw.github.com/uhsogs/dotfiles/master/setup.sh)"
```

## Initialize
for macOS

```bash
$ ~/dotfiles bash init_mac.sh
```

## Deploy dotfiles

```bash
$ ~/dotfiles bash deploy.sh
```
